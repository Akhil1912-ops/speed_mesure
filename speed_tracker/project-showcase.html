Campus Speed Tracker

Flutter mobile app with automated route verification for campus vehicle monitoring

What it does

Records vehicle trips via GPS, enforces a 30 km/h campus speed limit, verifies routes against approved paths, and generates a QR code for security to scan at the exit gate.

Flow

1. Record trip → 2. Upload to Firebase → 3. Auto-verify route → 4. Get QR code → 5. Security scans & decides

Tech stack

Flutter Firebase Firestore Cloud Functions Firebase Storage OpenStreetMap

Features

Real-time GPS speed tracking, color-coded route map (green/orange/red by speed), buffer-based route matching, penalty calculation, and a web dashboard for security review.
